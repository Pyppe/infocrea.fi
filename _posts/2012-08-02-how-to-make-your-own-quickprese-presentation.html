--- layout: post title: "How to make your own QUICKprese presentation" date: 2012-08-02T13:31:48+03:00 categories: general excerpt: |

Simple seminar case: take photos and record audio simultaneouslly.

  1. Create QUICKprese account and combine it with a Flickr user account.  Flickr is essential so create Flickr account first if there is none yet.
  2. When seminar presentation starts take the first photo and start audio recording simultaneously.
  3. Upload photos to Flickr account above.
  4. Start a new presentation in QUICKprese.
  5. Upload audio file.
  6. Audio start time = the time of the first photo uploaded to Flickr. (Actions -> Edit title, tags and dates.)
  7. Wait a while so that audio files and photos are certainly loaded.
  8. Start viewing and listening the presentation.
